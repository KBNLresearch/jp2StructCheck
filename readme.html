Command line synopsis (see documentation for details)

usage: jp2StructCheck.py [-h] [-v] [-t] jp2In

positional arguments:

 jp2In:  path to input JP2 image(s)

optional arguments:

 -h, --help:    show this help message and exit

 -v, --version:    show program's version number and exit

  -t:    report output in terse format
